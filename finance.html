CS50 Finance App

This is the CS50 Finance app, a web application built using Flask. It allows users to buy and sell stocks, manage their portfolio, and track stock prices in real-time.

Features:

  • User authentication: Users can register and log in securely.
  • Stock lookup: Users can search for stock symbols and get real-time stock prices.
  • Buying stocks: Users can purchase stocks using virtual money.
  • Selling stocks: Users can sell stocks from their portfolio.
  • Portfolio management: Users can view their current holdings and their total portfolio value.
  • Transaction history: Users can view their transaction history including buy and sell transactions.

This application is built as part of the CS50 course and serves as a demonstration of web development using Flask.
